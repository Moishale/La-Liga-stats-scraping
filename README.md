 # La Liga Scraper
Simple tool to scrape La Liga the Spanish league data
 
 ## Installation

```bash
cd Desktop
git clone https://github.com/Moishale/La-Liga-stats-scraping.git
python3 -m pip install -r requirments.txt
python main.py
```
